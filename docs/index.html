Oriz Pincode API

Indian PIN code → district + state. Monthly snapshot of data.gov.in.

Endpoints

  • /data/latest.json — current snapshot
  • /data/pincodes.json — same data, stable name

Both are flat JSON: { source, fetchedAt, count, pincodes: [{ pincode, district, state, officeName }] }.

Schedule

Refreshed monthly on the first Monday at 06:30 IST. Workflow re-fetches the upstream CSV; if upstream is unreachable, the last good snapshot is retained.

Data source: data.gov.in — All India Pincode Directory, used under the Government Open Data License – India. Code MIT. Source on GitHub.
